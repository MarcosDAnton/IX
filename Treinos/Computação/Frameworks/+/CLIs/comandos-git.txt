Servem para realizar o versionamento de código de pastas e arquivos de pastas na plataforma GitHub pelo Bash, CMD ou Terminal


git clone link-do-repositorio 
(Navegue a até a pasta desejada para clonar um repositório)





git add. 
git commit -m "data-do-commit" 
git push origin main ou outra-branch

(Essa é a sequência para selecionar todos os arquivos de um repositório e enviar do local para o GitHub)

Correção em caso de error: 

git config --global user.name "Seu Nome"
git config --global user.email "seu-email@example.com"
